# 阪元さん（Sakamoto-san）

Slack bot powered by node.js and Botkit.

## Dependency

* node
* npm (and yarn)

## Hot to use

* Slackのtokenを発行: http://www.utakata.work/entry/2017/11/18/100000
* `.env.sample` をコピーして `.env` を作成
* `yarn install`
* `yarn run start`
